# It All Leads to Doom

A fan-made Next.js countdown site for Marvel Studios' *Avengers: Doomsday* release on **December 18, 2026**.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

To build and run production locally:

```bash
npm run build
npm run start
```

## Core Capabilities

- Live countdown timer to December 18, 2026
- Animated hero layout with cinematic poster background
- Character teaser link cards (YouTube)
- Responsive design for mobile and desktop

## Configuration

No environment variables are required.

Project commands:

- `npm run dev` - start local development server
- `npm run build` - create production build
- `npm run start` - run production server
- `npm run lint` - run ESLint

## Usage

1. Start the app with `npm run dev`.
2. View the live countdown on the homepage.
3. Click trailer cards to open teaser videos in a new tab.
